v1.11.0
- `common/ui_base.py`: add `_poll(check_fn, timeout)` helper — polls every 1s up to timeout; timeout <= 0 runs once
- All page objects `is_visible()`: add `timeout=0` parameter; delegates to `_poll` so callers can block until visible without writing their own loop
- `common/ui_mtr.py`: fix `dump_ui()` — use `_adb_raw` for both the dump and cat calls so transient `uiautomator dump` failures return `""` instead of raising
- `test_mtr_camera.py`: add 5s wait before screenshot to let in-call UI settle; verify invite dialog is gone after dismiss; simplify step 3 and step 5 polling loops to use `is_visible(timeout=...)`

- Add `common/ui_mtr.py` (`MtrUi`): ADB-based UI controller with tap/swipe/keyevent, `dump_ui`, `find_element`, `tap_element`, `wait_for_element`, `current_activity`, app lifecycle, and lazy page-object properties
